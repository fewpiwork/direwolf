- Clean and Convenient API
- Simple to Set Headers, Cookies, Parameters, Post Forms
- Sessions Control
- Keep-Alive & Connection Pooling
- HTTP(S) Proxy Support
- Redirect Control
- Timeout Control
- Support extract result from response body with css selector, regexp, json
- Content Decoding
